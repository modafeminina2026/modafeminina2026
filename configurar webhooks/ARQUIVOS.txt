🚀 COMO COMEÇAR
═══════════════════════════════════════════════════════════════

1️⃣  Obter o Webhook Signing Secret:
    $ stripe login
    $ stripe listen --forward-to localhost:4242/webhook
    
    Copie o secret (começa com whsec_)

2️⃣  Configurar o Secret:
    $ npm run setup
    
    Ou edite o .env manualmente

3️⃣  Iniciar o Servidor:
    $ npm start
    
    Você verá: 🚀 Webhook server running on port 4242

4️⃣  Testar com Stripe CLI (em outro terminal):
    $ stripe trigger payment_intent.succeeded
    
    Veja os logs no servidor!
