# Edit Existing File

## BEFORE editing:
1. READ the entire file first
2. IDENTIFY exactly what needs to change
3. VERIFY changes will not break other areas
4. CHECK imports depending on this file

## DURING editing:
- Make ONLY necessary changes
- DO NOT rewrite entire file
- DO NOT remove working code
- DO NOT alter unrelated formatting
- KEEP used imports

## IF UNSURE:
- ASK before editing
