# Product Steering — Lover's Brasileiras

## IDENTITY
- Website name: Lover's Brasileiras
- Target audience: Men buying gifts for girlfriends/wives
- Categories: Lingerie, Perfumes, Jewelry, Makeup
- Tone: Romantic, sophisticated, luxurious, feminine
- Interface language: Brazilian Portuguese (pt-BR)
- Currency: BRL (Brazilian Real)

## ⚠️ CRITICAL BUSINESS DATA
- 80% of users access through mobile devices (Android and iOS)
- The website MUST be designed FIRST for mobile, THEN adapted for desktop
- If something looks good on desktop but bad on mobile, it is WRONG
- If something looks good on mobile but simple on desktop, it is ACCEPTABLE

## THEME COLORS
- Primary (pink): rose-500, rose-600, pink-500
- Secondary (purple): purple-400, purple-500, violet-400
- Support: white, gray-50, gray-100
- Text: gray-800, gray-900
- Error: red-500
- Success: green-500

## CONTENT RULES
- ALL visible text must be in pt-BR
- Prices must always use R$ with two decimal places
- Romantic but professional tone
- NEVER offensive or explicit content
- Images must be elegant and sophisticated

## PRODUCT FEATURES
- Persistent shopping cart
- Wishlist
- Personalized gift message
- Gift wrapping option
- Saved cards for fast checkout
- Filters by category, price, size, color
- Featured products

## REQUIRED MOBILE EXPERIENCE
- Navigation using bottom tabs or accessible hamburger menu
- Cart accessible with one tap (fixed icon in header)
- Large and always visible "Buy" button (minimum 48px height)
- Simplified step-by-step checkout
- Images optimized for fast loading on 4G/3G
- Swipe between product images
- Pull-to-refresh where appropriate
- Infinite scroll or "Load More" instead of numeric pagination
